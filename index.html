How to Use This Tool

This tool makes it harder to throw games by showing you a hand-curated list of the most important cards to play around, by region and mana cost. At the top you can select the regions you're playing against and the mana your opponent has to filter the list.

Want to fight about which cards should be here? Have a suggestion for this site? Have thoughts on the hint text? File an issue or pull request on this page's GitHub.
